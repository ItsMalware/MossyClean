# 🌿 Mossy Clean

**The Ultimate Glassmorphic Industrial Cleanup Tool for macOS.**

Mossy Clean is an industrial-grade duplicate file finder and organizer designed with a premium "Glassmorphic Pro" aesthetic. It combines bit-perfect SHA-256 discovery with a "Code Protection Engine" to keep your development environment safe and tidy.

![Mossy Mascot](https://github.com/ItsMalware/MossyClean/raw/main/.agent/agent.png)

## 💎 Features

- **SHA-256 Precision**: Identifies identical duplicates with 100% accuracy (no "similar" guessing).
- **Code Protection Engine**: Automatically recognizes and shields `.py`, `.swift`, `.js`, and other code files from accidental deletion.
- **Glassmorphic Pro UI**: A stunning interface built with native macOS `ultraThinMaterial` and industrial metallic borders.
- **Micro-Animations**: Features spring-based interactions and a "Living Industrial Mesh" backdrop.
- **Mossy Mascot**: A context-aware groundskeeper companion that provides real-time status reports.
- **Smart Organizer**: Automatically renames and filters files using professional industrial standards.

## 🚀 Getting Started

### Prerequisites
- macOS 13.0+
- Swift 5.9+

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ItsMalware/MossyClean.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MossyClean
   ```
3. Run the application:
   ```bash
   swift run
   ```

## 🛡️ Safety First

Mossy Clean is built for experts. The **Expert Risk Override** flow ensures that even if you need to delete protected code artifacts, you must explicitly confirm the risk or export the file path for inspection first.
